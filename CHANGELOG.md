# CHANGELOG

## [1.0.0] — 2026-07-20

### 新增功能

#### 番茄钟
- 标准番茄工作法：25 分钟专注 / 5 分钟短休 / 15 分钟长休
- SVG 进度环实时动画
- Web Worker 高精度计时 + 状态机
- 浏览器桌面通知 + 音效提醒
- 可自定义专注/休息时长

#### 习惯打卡
- 习惯 CRUD（名称/图标/颜色/目标天数）
- 每日快捷打卡 + 取消打卡
- Dashboard 一键打卡面板
- 拖拽排序（持久化到后端）
- 连续打卡统计（streak）

#### 任务管理
- 任务 CRUD（标题/优先级/预估番茄数）
- TODO → DOING → DONE 状态循环切换
- 拖拽排序持久化
- 番茄钟关联（任务番茄计数）
- Tab 筛选（全部/待办/进行中/已完成）

#### 数据可视化
- 仪表盘四维数字卡片（习惯/打卡/专注/任务）
- 本周打卡热力图（7 天 x N 习惯）
- 番茄钟趋势柱状图（周/月）
- SVG 贝塞尔平滑折线图（Catmull-Rom → Cubic Bezier, tension=0.3）
- 4 档时间范围：7天 / 14天 / 30天 / 90天

#### 主题系统
- 深色/浅色主题切换
- 跟随系统自动切换
- CSS Variables 全局主题变量
- 热力图/折线图暗色适配

#### 数据管理
- JSON 格式完整导出（习惯 + 打卡 + 番茄 + 任务）
- JSON 格式完整导入（四阶段按序导入，已存在跳过）
- 数据导入结果统计（成功/跳过/失败）

#### 用户体验
- 5 种骨架屏变体（卡片/列表/图表/表格/文本）
- 键盘无障碍导航（focus-visible + ARIA 语义 + skip link）
- 全局错误处理 + 重试按钮
- API 请求超时控制（15s）
- 路由级懒加载

#### 工程化
- 一键构建脚本 (`build.ps1`)
- Docker Compose 开发环境
- 完整部署文档 (`DEPLOY.md`)
- API 接口文档 (`API.md`)
- E2E 集成测试规范
- 代码审查报告 (Week 2-6)

### 技术栈
- 后端：Spring WebFlux + Halo Plugin SDK
- 前端：Vue 3 + TypeScript + Vite 5
- 图表：自研 SVG 折线图 + ECharts
- 番茄钟：Web Worker + 状态机

## 版本说明

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

- **主版本号 (MAJOR)**：不兼容的 API 修改
- **次版本号 (MINOR)**：向下兼容的功能新增
- **修订号 (PATCH)**：向下兼容的问题修正

`1.0.0` 为首次正式发布，包含插件所有核心功能。
